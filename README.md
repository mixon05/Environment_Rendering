# GKOM_Environment_Rendering

## Uwagi

1) Używać float zamiast double
2) Fajna książka: https://learnopengl.com/

# Ruch

W/S - przód/tył
A/D - lewo/prawo
Spacja (+shift) - góra/dół

Strzałki - obrót

## Linux

$ cd gkom_environment_rendering
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./gkom_enviroment_rendering
